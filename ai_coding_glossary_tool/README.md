# AI 编程小白词典

这是一个本地 Windows 小工具，读取你的 Markdown 词库，不联网、不接入 API。

## 用法

1. 在任意 App 或浏览器里选中术语，例如 `push origin`、`DNS`、`API Key`。
2. 按 `Ctrl+Shift+E`。
3. 小窗口会弹出“小白版解释、常见场景、操作/风险提醒”。

也可以打开窗口后：

- 在搜索框手动输入术语。
- 点“解释剪贴板”。
- 在窗口里右键，使用“解释刚才选中的文字 / 解释剪贴板 / 复制当前解释 / 重载词库”。

## 词库

默认读取项目内置词库：

```txt
data\glossary\ai_coding_beginner_glossary.md
```

后续你直接修改这份 Markdown，再点“重载词库”即可。

## 说明

- 这是本地词典版，不会把内容发到网络。
- 全局快捷键通过临时模拟 `Ctrl+C` 读取选中文字。工具会尽量恢复原来的文本剪贴板；如果剪贴板里原本是图片或复杂格式，Windows 可能无法完整恢复。
- “任意 App 自带右键菜单里新增解释项”需要更重的系统插件或分别给浏览器、编辑器做扩展。当前版本先提供全局快捷键和工具窗口右键菜单。

## 卸载

运行安装目录里的 `uninstall.ps1`。
